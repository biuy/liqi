随机回复

日常成精

发送群消息。每条消息有5%概率触发。

开关全局随机回复
随机回复与复读同步开关。
  • admin
  • repeat on
  • admin
  • repeat off
添加随机回复内容
添加内容到回复库。
  • admin
  • addreplycontent [内容]
 [内容]可以是文字、图片、表情。
例子
  • admin
  • addreplycontent 和谁
删除随机回复内容
从回复库中删除内容。
  • admin
  • delreplycontent [内容]
 [内容]是待删除的文字、图片、表情。
例子
  • admin
  • delreplycontent 和谁
